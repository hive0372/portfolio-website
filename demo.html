Portfolio Website Enhancements

I've made several enhancements to your portfolio website to make it more interactive and appealing to recruiters. Here's a summary of the changes:

1. Enhanced Contact Form

The contact form has been completely redesigned with:

  • Animated input fields that respond to hover and focus
  • Improved validation with visual feedback
  • A gradient submit button with shimmer effect
  • Animated success message with pulse effect
.contact__button { background: linear-gradient(90deg, var(--clr-primary), var(--clr-primary-light), var(--clr-primary)); background-size: 200% 100%; transition: all 0.3s ease; box-shadow: 0 4px 15px rgba(var(--clr-primary-rgb), 0.3); } .contact__button:hover { background-position: right center; transform: translateY(-3px); box-shadow: 0 7px 20px rgba(var(--clr-primary-rgb), 0.4); }

2. Interactive Project Cards

Project cards now feature:

  • Hover overlay with call-to-action
  • Animated image zoom effect
  • Interactive category badges
  • Animated link buttons with fill effect
.project__image-overlay { position: absolute; top: 0; left: 0; width: 100%; height: 100%; background: rgba(var(--clr-primary-rgb), 0.7); display: flex; justify-content: center; align-items: center; cursor: pointer; } .project__link::before { content: ''; position: absolute; top: 0; left: 0; width: 0%; height: 100%; background-color: var(--clr-primary); transition: all 0.3s ease; z-index: -1; } .project__link:hover::before { width: 100%; }

3. Project Filtering System

Added an interactive project filtering system:

  • Collapsible filter menu
  • Animated category buttons
  • Smooth transitions between filtered projects
  • Empty state handling with reset option

Additional Enhancements

  • Added RGB color variables to support rgba() usage in overlays and animations
  • Enhanced button animations with gradient backgrounds and hover effects
  • Improved form field interactions with subtle animations
  • Added animated success messages with pulse and shimmer effects

These enhancements make your portfolio more engaging and interactive, helping to capture recruiters' attention and showcase your projects in a more appealing way.

To see all the changes in action, you'll need to:

  1. Clone the repository locally
  2. Run npm install --legacy-peer-deps to install dependencies
  3. Run NODE_OPTIONS=--openssl-legacy-provider npm start to start the development server

You may need to resolve some dependency conflicts due to the version of React and the animation libraries used.
